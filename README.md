# ss14-ru-web

Сайт для проекта Space Station 14 RU от сообщества SS14 Devbus RU.

## Запуск

```bash
git clone https://github.com/ss14-art/ss14-ru-web.git
cd ss14-ru-web
npm install
```

### Dev

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview build

```bash
npm run preview
```
